Basic of Unitest framework that is inbuilt within Selenium

1. Create class with argument unittest.Testcase
2. All Test Names should start with ‘test_’
3. Def are created with args 'self'

Before and after methods in unittest
1. Setup
2. Teardown
3. SetupClass
4. TearDownClass
5. SetupMethod
6. TearDownMethod

See Example below: 


#called once, before anything else in this module
def setUpModule():        
      print ('Running module %s' % __name__)


#called once, after everything else in this module
def tearDownModule():    
    print ('Ran module %s' % __name__)

class UnitTestBasics(unittest.TestCase):
    @classmethod #This is reqd
    def setUpClass(cls):
        print("Executes before any Test method within the class runs")
    
    @classmethod
    def tearDownClass(cls):
        print("Executes after all Test method within the class runs")

    def setUp(self):
        print("This is the precondition to the method")

    def tearDown(self):
        print("This is the postcondition to the method")

Types of Assertions in Unitest

  • assertEqual
  • assettNotEqual
  • assertTrue
  • assertFalse
  • assertIn
  • assertNotIn
  • assertIsNone
  • assertIsNotNone


class AssertionsinUnittest(unittest.TestCase):

    #Example of assertEqual          
    def test_TestCase1(self):
        self.assertEqual("google", "google", "The page title doesnot match")
    
    #Example of assertNotEqual 
    def test_TestCase2(self):
        self.assertNotEqual("google", "google", "The page title matches")

    #Example of assertTrue 
    def test_TestCase3(self):
        self.assertTrue(5 == 6,"The page title doesnot match")

    #Example of assertTrue           
    def test_TestCase4(self):
        self.assertFalse(5 == 6, "The page title matches")
    
    #Example of assertIsNone 
    def test_TestCase5(self):
        xy=5
        self.assertIsNone(xy,"Var is not None")

    #Example of assertIsNotNone 
    def test_TestCase6(self):
        xy=None
        self.assertIsNotNone(xy,"Var is not None")

    #Example of assertIn 
    def test_TestCase7(self):
        lst ={'xy','xb','xa'}
        self.assertIn('xz',lst,"Var not is not lst")

    #Example of assertNotIn 
    def test_TestCase8(self):
        lst ={'xy','xb','xa'}
        self.assertNotIn('xy',lst,"Var not is in lst")

4. To run there are 2 ways Tests in Unitest:
i. Using 'main' 


if __name__="__main__":
    unittest.main()

ii. Running Tests in Groups
First Import Test Class from different packages 


#First Import Test Class from different packages
from Package1.className import TestName

#Create Test Groups
TC_group_1= unittest.TestLoader().loadTestsFromTestCase(classnames1)

#Please Note: All Test methods under className1 will be added here.
TC_group_2= unittest.TestLoader().loadTestsFromTestCase(classnames2)

#After the Groups are added, Create suites from Groups
Suite1 =unittest.TestSuite([TC_group_1, TC_group_2])

#Run Suite
unittest.TextTestRunner().run(Suite1)
